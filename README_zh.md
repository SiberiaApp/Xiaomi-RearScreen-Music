# 小米背屏音乐助手
> 适用于小米背屏系列的音乐播放器。

*[English](README.md) | [简体中文](README_zh.md)*

专为配备背屏/副屏的小米机型量身定制的动态音乐播放器小部件。深度结合 HyperOS/MIUI 的小部件特性，为您带来丝滑、原生的音乐控制体验。

### ✨ 核心功能
* **平滑的歌词动效**：当歌词或歌曲标题更新时，文本会以丝滑的底部上浮动效呈现，彻底告别原生的闪烁问题。
* **自适应专辑色彩**：支持 HyperOS 动态材质（HyperMaterial）与调色板提取，背景和光晕会根据当前播放的专辑封面自动改变颜色，带来沉浸式视觉体验。
* **完美的息屏显示 (AOD) 适配**：在亮屏与息屏状态之间无缝切换。进入 AOD 时会自动调整排版、透明度与动画，既省电又能保证信息直观可见。
* **完整的媒体控制**：包含播放、暂停、上一首、下一首的完整控制面板，并且按钮内置了流畅的 Lottie 交互动画。
