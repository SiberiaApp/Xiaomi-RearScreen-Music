# Xiaomi RearScreen Music
> About: A music player compatible with the Xiaomi RearScreen series.

A highly dynamic, elegantly designed music player widget custom-built for Xiaomi devices equipped with a rear screen. It leverages HyperOS/MIUI widget capabilities to deliver a seamless, native-feeling audio control experience.

### ✨ Features
* **Dynamic Lyric Animations**: Lyrics slide up smoothly with a custom transition effect whenever the track progresses, without visual stuttering.
* **HyperMaterial & Adaptive Colors**: Automatically extracts color palettes from the current album art to tint the background and UI elements, creating an immersive visual experience.
* **Always-On Display (AOD) Support**: Seamlessly transitions between active screen and AOD states, rearranging the layout (like artist and title) to save power while keeping information visible.
* **Full Media Controls**: Easily play, pause, skip, or go back to previous tracks with smooth Lottie icon animations.
